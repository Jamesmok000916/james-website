Self Introduction

Hello! I am Mok Ching Hei, you can call me James, I am currently studying Bachelor of Science (Hons) Degree in Information Security in The Hong Kong Polytechnic University (PolyU).

What fascinates me to study Information Security is because I realize that nowadays technologies bring convenience to our daily life, but at the same time, it may contain many security issues. This makes me increase my awareness about the information security problems. So I hope I can learn deeper about the knowledge and technologies behind the information security, like how to prevent security risks, understand the way to protect the data, analysis and solve the security problems.

In the future, I hope I can contribute to the IT industry as the demand for IT is increasing. According to the 2019 Robert Half Salary Guide, it shows that it is challenging to find qualified IT staff in IT security. So that I want to contribute to the IT industry especially in Information Security like internal/external security auditor or information security officer.
